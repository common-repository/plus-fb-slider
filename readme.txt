Plus Facebook Slider is Responsive Facebook Slider Sidebar Plugin.

== Description ==

Plus Facebook Slider lets you easily embed and promote any Facebook Page on your website sidebar. 

Facebook Like Box takes a lot of space but as our widget is positioned on absolute area you can put your facebook likebox without worrying about space.

Just like on Facebook, your visitors can like and share the Page without leaving your site.

You can look at demo here: http://twitterslider.com/wp/

== Installation ==

Thanks for installing or downloading our Pinterest Widget

1. Install the wordpress plugin first
You can install it via Wordpress Installer or can download the file first then Upload it on 'wp-content' folder

2. Activate the plugin

3. You will see facebook settings from right side of your wordpress menu.

Plus Facebook Slider Settings:

Plus Facebook Slider Allow You To Display Facebook Profile In Your Website,

1. Add Nmae                         (example = facebook)
2. AFacebook Page URL               (example = https://www.facebook.com/Facebook)
3. Add Margin Top                   (example = 150)
4. Add Alignment Position           (example = left)
5. Add Display Face                 (example = true)
6. Add Display Post                 (example = true)
7. Add Hide Cover Photo             (example = false)
8. Add Language                     (example = english)
